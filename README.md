### Platforming:

#### Keyboard:
Laufen: A und D, oder Pfeil Rechts und Links

Ducken: C

#### Controller:
Laufen: Linker Joystick
Springen: Button South (z.B. A bei Xbox, X bei Playstation)
Ducken: Linker Trigger
